Coloque aqui os PNGs dos moveis estilizados.

O app tenta carregar primeiro os PNGs desta pasta. Se nao encontrar, usa o desenho nativo.

Use nomes pelo codigo do item quando puder:

- quarto/cama_casal.png
- saladeestar/sofa.png
- cozinha/mesa_cozinha.png
- banheiro/vaso.png
- geral/cadeira.png

Tambem funciona com alguns nomes sem acento e sem espaco, como:

- quarto/camacasal.png
- quarto/camacasal1.png
- quarto/camadecasal1.png
- saladeantar/mesa1.png
